title: "Latexmk の導入"
---

## Latexmk とは

LaTeX では相互参照や目次を正確に生成するために同じファイルを複数回コンパイルする必要があります．参考文献がある場合はさらに BibTeX を挟む必要があります．Latexmk はこういった煩雑な手順を自動で判断して実行してくれるツールです．`texlive/texlive` イメージには最初から含まれています．

## 設定ファイルの作成

はじめに Latexmk の設定ファイルを作ります．プロジェクトフォルダに `.latexmkrc` という名前のファイルを作成してください．ファイルの中身は次のようにしてください．

```perl
$pdf_mode = 4;

$lualatex = 'lualatex -synctex=1 -interaction=nonstopmode %O %S';
$bibtex = 'upbibtex %O %B';
```

各項目について簡単に説明します．

`$pdf_mode` は LaTeX エンジンの出力形式を指定するための変数です．LuaLaTeX を使う場合は 4 を指定します．

`$lualatex` は LuaLaTeX でコンパイルする際のコマンドを指定するための変数です．`-synctex=1` は SyncTeX を有効にするオプションで，PDF と TeX ソースの位置を対応付けるための設定です．`-interaction=nonstopmode` はエラーが発生してもコンパイルを続行するオプションです．`%O` と `%S` は Latexmk が自動で置き換えてくれるプレースホルダです．

`$bibtex` は BibTeX で参考文献を処理する際のコマンドを指定するための変数です．今回は日本語のドキュメントなので `upbibtex` を指定しています．`%O` と `%B` は Latexmk が自動で置き換えてくれるプレースホルダです．

## Latexmk を用いたコンパイル

設定ファイルを作成したら，実際に Latexmk を使ってコンパイルします．ターミナルで以下のコマンドを実行してください．

```bash
latexmk main.tex
```

しばらくするとコンパイルが完了し，`main.pdf` が生成されます．エクスプローラーから `main.pdf` を開いて中身を確認してください．

## 別の LaTeX エンジンへの変更

LuaLaTeX 以外の LaTeX エンジンに変更したい場合は，`.latexmkrc` の内容を変更します．以下にいくつかの例を示します．必要に応じて利用してください．

### pdfLaTeX

```perl
$pdf_mode = 1;

$latex = 'pdflatex -synctex=1 -interaction=nonstopmode %O %S';
```

### upLaTeX

```perl
$pdf_mode = 3;

$latex = 'uplatex -synctex=1 -interaction=nonstopmode -file-line-error %O %S';
$bibtex = 'upbibtex %O %B';
$dvipdf = 'dvipdfmx %O -o %D %S';
```

### pLaTeX

```perl
$pdf_mode = 3;

$latex = 'platex -synctex=1 -interaction=nonstopmode -file-line-error %O %S';
$bibtex = 'pbibtex %O %B';
$dvipdf = 'dvipdfmx %O -o %D %S';
```

## 設定ファイルのシンタックスハイライトの有効化

Latexmk によるコンパイルが完了したら，次は設定ファイルのシンタックスハイライトを有効にします．`.vscode/settings.json` というファイルを作成してください．ファイルの中身は次のようにしてください．

```json
{
  "files.associations": {
    ".latexmkrc": "perl"
  }
}
```

以上で Latexmk 導入は完了です．
