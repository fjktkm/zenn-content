## VS Code とは

Visual Studio Code（VS Code）は，Microsoft が開発したテキストエディタです．拡張機能が豊富で，さまざまなプログラミング言語やツールに対応しているため，ソフトウェア開発の現場で広く利用されています．現在世界でもっともプログラマに利用されているエディタです．

## VS Code のインストール

VS Code をインストールします．すでにインストール済みの場合は，このセクションを読み飛ばしてください．推奨するインストール手順は次の通りです．

1. 以下のページを開き「Microsoft Store で見る」ボタンをクリックする
2. Microsoft Store アプリを起動するかどうかを確認するポップアップが表示された場合は承認する
3. Microsoft Store アプリの VS Code のページが開いたら，「インストール」ボタンをクリックしてインストールを実行する

https://apps.microsoft.com/detail/XP9KHM4BK9FZ7Q?hl=ja&gl=JP&ocid=pdpshare

## VS Code の起動

インストールが完了したら，次に VS Code を起動します．推奨する起動手順は次の通りです．

1. Windows キーを押してスタートメニューを開く
2. 検索バーに「code」と入力する
3. 検索結果の「Visual Studio Code」の項目をクリックして起動する

## Remote Development 拡張機能のインストール

VS Code のインストールが完了したら，次に Remote Development 拡張機能をインストールします．Remote Development 拡張機能がすでにインストール済みの場合は，このセクションを読み飛ばしてください．推奨するインストール手順は次の通りです．

1. 以下のページを開き「Install」ボタンをクリックする
2. VS Code を起動するかどうかを確認するポップアップが表示された場合は承認する
3. VS Code が起動し Remote Development 拡張機能のページが開いたら，「Install」ボタンをクリックしてインストールを実行する

https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack

以上で VS Code のセットアップは完了です．使用したブラウザのウィンドウなどはすべて閉じて構いません．
