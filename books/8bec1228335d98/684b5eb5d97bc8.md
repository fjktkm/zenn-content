title: "BibTeX による文献管理"
---

## BibTeX とは

BibTeX は LaTeX で参考文献を管理するためのツールです．参考文献の情報を Bib ファイルに記述しておくことで，本文中での引用と参考文献リストの生成を自動化できます．`texlive/texlive` イメージには最初から含まれています．

## Bib ファイルの作成

はじめに，参考文献の情報を記述する Bib ファイルを作成します．TeX ファイルと同じ階層に `reference.bib` というファイルを作成してください．ファイルの中身は次のようにしてください．

```bibtex
@misc{miyazawa1934polano,
  author       = {宮沢 賢治},
  title        = {ポラーノの広場},
  year         = {1934},
  publisher    = {青空文庫},
  howpublished = {\url{https://www.aozora.gr.jp/cards/000081/files/1935_19925.html}}
}
```

各項目について簡単に説明します．

`@misc` は文献の種類を表します．ここでは雑多な文献を表す `@misc` を使用していますが，ほかにも論文を表す `@article`，会議論文を表す `@inproceedings`，書籍を表す `@book` などがあります．

`{miyazawa1934polano}` は文献のキーです．本文中でこの文献を引用する際に利用します．

`author`，`title`，`year`，`publisher`，`howpublished` は，それぞれ文献の著者，タイトル，発表年，出版社，入手方法を指定するフィールドです．ほかにも様々なフィールドがあります．

## TeX ファイルの編集

Bib ファイルの作成が完了したら，次に TeX ファイルに参考文献を引用するための処理を追加します．前のチャプターで作成した `main.tex` を次のように書き換えてください．

```latex
\documentclass{jlreq}

\begin{document}

\title{ポラーノの広場}
\author{宮沢賢治}
\maketitle

\section{はじめに}

あのイーハトーヴォのすきとおった風，夏でも底に冷たさをもつ青いそら，うつくしい森で飾られたモリーオ市，郊外のぎらぎらひかる草の波。\cite{miyazawa1934polano}

\bibliographystyle{junsrt}
\bibliography{reference}

\end{document}
```

各項目について簡単に説明します．

`\cite{キー}` は参考文献を引用するコマンドです．`キー` には Bib ファイルで定義した文献のキーを指定します．

`\bibliographystyle{スタイル}` は参考文献リストのスタイルを指定するコマンドです．ここでは `junsrt` を指定しています．ほかにも様々なスタイルがあり，英語であれば `plain` や `IEEETran`，日本語であれば `jplain` や `jalpha` などがあります．

`\bibliography{ファイル名}` は参考文献リストを出力するコマンドです．`ファイル名` には Bib ファイルの名前（拡張子なし）を指定します．

## BibTeX を用いたビルド

`reference.bib` の作成が完了したら，BibTeX を使ってビルドします．参考文献を正しく処理するには，以下の順番でコマンドを実行する必要があります．

```bash
lualatex main.tex
upbibtex main
lualatex main.tex
lualatex main.tex
```

まず `lualatex main.tex` を実行すると，`main.aux` というファイルが生成されます．次に `upbibtex main` を実行すると，`main.aux` を参照して参考文献リストを処理し，`main.bbl` というファイルが生成されます．最後に `lualatex main.tex` を 2 回実行することで，参考文献リストと引用番号が正しく反映された PDF が生成されます．

以上で BibTeX による文献管理の設定は完了です．
