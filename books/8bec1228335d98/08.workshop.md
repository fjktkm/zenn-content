title: "LaTeX Workshop の導入"
---

## LaTeX Workshop とは

LaTeX Workshop は，VS Code 上で LaTeX を快適に利用するための拡張機能です．前の章ではターミナルから手動で `latexmk` を実行しましたが，LaTeX Workshop を使うとファイルを保存するだけで自動的にコンパイルが走るようになります．また，VS Code やブラウザで PDF をプレビューする機能も提供しています．

## LaTeX Workshop Feature の追加

はじめに Dev Container の構成ファイル `devcontainer.json` を以下のように変更します．

```json
{
    "name": "LaTeX (TeX Live)",
    "image": "texlive/texlive:latest",
    "features": {
        "ghcr.io/devcontainers/features/common-utils:2": {}
    },
    "customizations": {
        "vscode": {
            "extensions": [
                "James-Yu.latex-workshop"
            ]
        }
    }
}
```

追加したのは `customizations` のブロックです．`extensions` に拡張機能の ID を記述すると，コンテナ起動時に自動でインストールされます．

構成ファイルの編集が完了したら，コンテナをリビルドしてください．

## `settings.json` の編集

コンテナのリビルドが完了したら，次に VS Code 拡張機能の設定ファイル `settings.json` を編集します．以下の内容に `settings.json` を書き換えてください．

```json
{
    "files.associations": {
        ".latexmkrc": "perl"
    },
    "latex-workshop.latex.recipe.default": "latexmk (latexmkrc)",
    "latex-workshop.formatting.latex": "latexindent",
}
```

各項目について簡単に説明します．

`"latex-workshop.latex.recipe.default": "latexmk (latexmkrc)"` は，LaTeX Workshop にあらかじめ用意されている Latexmk レシピを使う設定です．このレシピは，コンパイルする TeX ファイルと同階層にある `.latexmkrc` を参照して `latexmk` を実行します．前の章で記述した `.latexmkrc` をそのまま活かすことができます．

`"latex-workshop.formatting.latex": "latexindent"` は，LaTeX ファイルの自動フォーマットに `latexindent` を使う設定です．`texlive/texlive` イメージには `latexindent` が含まれているため，この設定を追加するだけで LaTeX ファイルをフォーマットできるようになります．

## 自動コンパイルの動作確認

`settings.json` の編集が完了したら，LaTeX Workshop による自動コンパイルが正しく動作するか確認します．`main.tex` の内容を適当に変更してファイルを保存してみてください．

コンパイルのステータスは VS Code の下部に表示されます．コンパイル中は「↻ Build...」と表示され，完了すると「✓」と表示されます．ファイルを保存するたびにコンパイルが走ることを確認してください．

## PDF のプレビュー

自動コンパイルが正しく動作していることを確認したら，次は PDF のプレビュー機能を確認します．まず VS Code 内で PDF をプレビューする方法について説明します．手順は次の通りです．

1. プレビューしたい TeX ファイルを開く
2. VS Code のウィンドウの左側にある「LaTeX」アイコンをクリックする
3. 「コマンド > LaTeX PDF を表示」を選択する

また，以下の手順でも同様に PDF をプレビューすることができます．

1. プレビューしたい TeX ファイルを開く
2. タブの右上にある「LaTeX PDF ファイルを表示」ボタンをクリックする

次にブラウザで PDF をプレビューする方法について説明します．手順は次の通りです．

1. プレビューしたい TeX ファイルを開く
2. VS Code のウィンドウの左側にある「LaTeX」アイコンをクリックする
3. 「コマンド > LaTeX PDF をブラウザで表示 > Web ブラウザで表示」を選択する

PDF のプレビューは，TeX ファイルを編集する度に自動でリロードされます．プレビューを開いた状態で TeX ファイルを編集して，プレビューが自動で更新されることを確認してください．

以上で LaTeX Workshop の導入は完了です．
