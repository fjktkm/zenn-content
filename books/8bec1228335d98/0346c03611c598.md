## WSL とは

Windows Subsystem for Linux（WSL）とは，Windows 上で Linux を動かすための仕組みです．

## Windows Terminal のインストール

はじめに Windows Terminal をインストールします．Windows Terminal がすでにインストール済みの場合は，このセクションを読み飛ばしてください．推奨するインストール手順は次の通りです．

1. 以下のページを開き「Microsoft Store で見る」ボタンをクリックする
2. Microsoft Store アプリを起動するかどうかを確認するポップアップが表示された場合は承認する
3. Microsoft Store アプリの Windows Terminal のページが開いたら，「インストール」ボタンをクリックしてインストールを実行する

https://apps.microsoft.com/detail/9N0DX20HK701?hl=ja-jp&gl=JP&ocid=pdpshare

## PowerShell のインストール

Windows Terminal のインストールが完了したら，次に PowerShell をインストールします．PowerShell がすでにインストール済みの場合は，このセクションを読み飛ばしてください．推奨するインストール手順は次の通りです．

1. 以下のページを開き「Microsoft Store で見る」ボタンをクリックする
2. Microsoft Store アプリを起動するかどうかを確認するポップアップが表示された場合は承認する
3. Microsoft Store アプリの PowerShell のページが開いたら，「インストール」ボタンをクリックしてインストールを実行する

https://apps.microsoft.com/detail/9MZ1SNWT0N5D?hl=neutral&gl=JP&ocid=pdpshare

:::message
なお，「PowerShell」と「Windows PowerShell」は別のアプリケーションです．「Windows PowerShell」はレガシーなバージョンであり，現在は「Windows」の付かない無印の「PowerShell」を使用することが推奨されています．必ず無印の「PowerShell」であることを確認してください．
:::

## PowerShell を管理者権限で起動する

PowerShell のインストールが完了したら，次に Windows Terminal を管理者権限で起動します．推奨する起動手順は次の通りです．

1. Windows キーを押してスタートメニューを開く
2. 検索バーに「wt」と入力する
3. 検索結果の「Windows Terminal」の項目の中から「管理者として実行」をクリックする
4. ユーザーアカウント制御のダイアログが表示された場合は「はい」をクリックして承認する

基本的には Windows Terminal を起動すると PowerShell が起動するはずですが，もしコマンドプロンプトや Windows PowerShell が起動した場合は，タブの右側にある「∨」ボタンをクリックして表示されるリストの中から「PowerShell」を選択して起動してください．

また，デフォルトのシェルが PowerShell になっていない場合は，変更しておくとよいでしょう．デフォルトのシェルを変更するには，タブの右側にある「∨」ボタンをクリックして表示されるリストの中から「設定」を選択し，「スタートアップ > 既定のプロファイル」の項目で「PowerShell」を選択して保存してください．

## WSL のインストール

PowerShell を管理者権限で起動したら，次に WSL をインストールします．以下のコマンドを実行してください．

```powershell
wsl --install
```

問題が発生した場合は，以下の公式ドキュメントのページを参考にしてください．

https://learn.microsoft.com/ja-jp/windows/wsl/install

## Linux の初期設定

WSL のインストールが完了すると，自動で Ubuntu のインストールと起動が実行されます．以下のような内容が表示されます．

```powershell
> wsl --install
ダウンロード中: Linux 用 Windows サブシステム 2.6.3
インストール中: Linux 用 Windows サブシステム 2.6.3
Linux 用 Windows サブシステム 2.6.3 はインストールされました。
この操作を正しく終了しました。
ダウンロードしています: Ubuntu
インストールしています: Ubuntu
ディストリビューションが正常にインストールされました。'wsl.exe -d Ubuntu' を使用して起動できます
Ubuntu を起動しています...
Provisioning the new WSL instance Ubuntu
This might take a while...
Create a default Unix user account: <あなたの Windows のユーザー名>
```

Ubuntu の起動が完了すると，Linux のユーザー名とパスワードの設定を求められます．ユーザー名については，特にこだわりがなければデフォルトのまま使用すれば問題ありません．Enter キーを押すとユーザー名が確定されます．

次にパスワードの設定を求められます．パスワードはセキュリティ上の理由から入力しても画面には表示されません．Enter キーを押すとパスワードが確定されます．確認のためにもう一度同じパスワードの入力が求められるので，もう一度同じパスワードを入力して Enter キーを押してください．

問題が発生した場合は，以下の公式ドキュメントのページを参考にしてください．

https://learn.microsoft.com/ja-jp/windows/wsl/setup/environment#set-up-your-linux-username-and-password

以上で WSL のセットアップは完了です．使用した Windows Terminal のウィンドウなどはすべて閉じて構いません．
